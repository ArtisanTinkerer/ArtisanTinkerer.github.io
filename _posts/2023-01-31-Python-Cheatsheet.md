# Python Data Structures

## List ```[1,2,3,4,5]```
* like an array
* allow duplicates

## Tuple `````(1,2,3,4,5)`````
* more like an array of constants
* allow duplicate elements

## Dictionary `````{1:"a", 2:"b"}`````
* ordered key-value pairs
* does not allow duplicate keys

## Set `````{1,2,3,4,5}`````
* will not allow duplicate elements
